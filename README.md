### 无头浏览器是模拟浏览器，所有失败原因皆获取自页面，返回的失败原因分为3种：

```
1.是页面或者标签里获得的失败原因，经星粒解释我们是会返回采购的，不做修饰
2.是页面或者标签里获得的失败原因，不宜返回采购的，稍作修饰。
3.是获取页面或者标签超时，做了修饰的
```

### 自己修饰的原因有以下情况

```
- 什么情况下修饰
    添加常旅遇到的12306返回的错误
    核验常旅的时候根据标签是未核验的
    页面打开超时或者标签查找超时
    12306返回错误的页面，标识Ip被封禁
    检查接口参数人数不对
    检查接口参数身份证ID小于14，这里包括了港澳，因没对接港澳
    检查接口参数传给的坐席码和12306页面不匹配
- 修饰成什么
    李浩@xxxx@证件号码输入有误!
    李浩@xxxxx@待核验
    系统繁忙，请稍后重试

    提交订单失败：乘客人数不符
    提交订单失败：证件格式不对
    座席类型不匹配
```
